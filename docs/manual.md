# Manual de marca J3F 2026 — Documento canônico

## Sobre

O **Manual de marca J3F 2026** é o documento de origem do brand kit. PDF de 60 páginas, 18 MB, entregue por **Lucas Barreira** em março de 2026.

Toda informação técnica deste repositório (tokens, regras de uso, paleta, tom de voz) é uma **derivação** do manual original. Em caso de divergência, o PDF é a autoridade final.

## Por que não está no repositório

18 MB é grande demais para versionar via Git tradicional (cada commit duplicaria o arquivo no histórico, inflando o repo rapidamente). Alternativas consideradas:

| Opção | Por que não escolhida |
|---|---|
| Commit direto | Repo cresceria várias centenas de MB ao longo do tempo |
| Git LFS | Adiciona dependência e custo de banda no GitHub |
| Compressão | PDF já está otimizado, ganho seria marginal |

**Decisão:** PDF permanece fora do repo, acessível por canais oficiais J3F.
